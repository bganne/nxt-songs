# Songs for playing on Lego NXT (Mindstorm)

This is a fork of https://github.com/robsoncouto/arduino-songs for Lego NXT (Mindstorm).

## Usage

The original Arduino code is compiled as C++ code to run on the host and generates the NXC code for Lego NXT.

Just run `make -j` to generate the relevant NXC code in each directory.
